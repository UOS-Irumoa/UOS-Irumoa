## 이루모아 - 서울시립대학교 비교과 정보 통합 및 AI 추천 서비스

## ✨ 이루모아 주요 기능
1. 흩어져 있는 시립대 비교과 정보를 **통합 제공**
2. 원하는 비교과를 빠르게 찾을 수 있도록 **검색 편이성 강화**
3. 비교과 **지원 자격**과 **카테고리**를 명확하게 보여주는 직관적인 UI
4. 관심사 기반 **개인 맞춤형 비교과 추천**
